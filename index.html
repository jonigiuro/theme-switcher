Image based themes

This piece of code reads the dominant color of the currently displayed image of the slideshow and then modifies some page elements accordingly (in this case background-color and title color).
It has a setTheme function which receives the rgb values, in there you can do whatever you want, making it easilly usable with any other slideshow, gallery or anything that works with images.
Please note that the canvas below needs to be there for the script to compute colors, but it's not mandatory that it's visible to the user. You can set its visibility to hidden or display:none.
